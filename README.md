# Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Smooth Scrolling**: Enhanced navigation experience
- **Interactive Elements**: Hover effects and transitions throughout
- **Contact Form**: Ready-to-use contact form (needs backend integration)
- **PDF Integration**: Links to your resume and work history PDFs

## Sections

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **About**: Personal information and statistics
3. **Skills**: Showcase of technical skills and technologies
4. **Projects**: Portfolio of featured projects
5. **Contact**: Contact information and form

## Customization

### Personal Information
1. Update the name in the hero section (line 30 in `index.html`)
2. Modify the subtitle and description to match your profession
3. Update the statistics in the About section
4. Add your actual skills in the Skills section
5. Replace placeholder projects with your real projects
6. Update contact information (email, LinkedIn, GitHub)

### Styling
- Primary color: `#6366f1` (defined in `:root` in `styles.css`)
- Secondary color: `#8b5cf6`
- All colors can be customized in the CSS variables section

### PDF Files
The website includes links to:
- `履歴.pdf` (Resume)
- `職務経歴書.pdf` (Work History)

Make sure these files are in the same directory as `index.html`.

## Getting Started

1. Open `index.html` in a web browser
2. Customize the content with your information
3. Replace placeholder images and project details
4. Update contact form to connect to your backend (if needed)
